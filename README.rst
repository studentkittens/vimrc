Installation
============

.. code-block:: bash

   $ git clone https://github.com/studentkittens/vimrc
   $ curl https://raw.githubusercontent.com/Shougo/neobundle.vim/master/bin/install.sh | sh
   $ cp .vimrc ~/
   $ mkdir ~/.vim/plugins
   $ mkdir ~/.vim/tmp/undo
   $ cp plugins/* ~/.vim/plugins/
